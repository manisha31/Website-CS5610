Awesome responsive layout using Bootstrap


Goal:

To create a responsive layout using Bootstrap and explore its different components.

Description:

I have illustrated a responsive design using Bootstrap in this experiment. The first component uses the class ".hero-unit". Then, I use a ".row" class to contain four boxes. To convert this to the fluid design, we have to use ".container-fluid" instead of ".container" class. Similarly we use ".row-fluid" instead of ".row" class. This is a very easy and efficient method to get responsive design and fluid grids. I also demonstrate the "Navigational Bar" containing a dropdown menu. In the end, I show a progress bar which is 70% complete.
